Как добавить новый виртуальный хост в Apache 2 на Ubuntu

  1. Скопировать в /etc/apache2/sites-available конфиг-файл нового сайта по образцу (default)
  2. Исправить новый конфиг-файл сайта
  3. sudo a2ensite newsite
  4. sudo /etc/init.d/apache2 restart
